03 / Components

Knapvarianter

Fire varianter dækker alle scenarier. Glassmorphism er forbeholdt sekundær (nav CTA-mønsteret); primær er solid orange.

Primær

Solid orange. Til den vigtigste handling på en flade — én pr. skærm.

Sekundær (glassmorphism)

Til navigation og sekundære CTAs på sand- eller orange-baggrund. Hover skifter til solid orange.

Ghost

Outline only. Til tertiære valg eller når flere knapper står ved siden af hinanden.

Disabled

Neutral grå med dæmpet tekst. Reagerer ikke på hover.

På mørk baggrund

Primær og ghost har inverterede states på Off Black-sektioner.
